# Welcome to Rust Blueprint

Rust Blueprint is a powerful CLI tool that generates production-ready Rust web applications in seconds. Whether you're building a REST API, GraphQL server, or full-stack application, Rust Blueprint sets up the perfect project structure so you can focus on writing business logic, not boilerplate.

## Quick Start

Get started with Rust Blueprint in just three steps:

```bash
# Install Rust Blueprint
cargo install rust-blueprint

# Create a new project
rust-blueprint create

# Run your application
cargo run
```

## Features at a Glance

### Web Frameworks
- Axum - Ergonomic and modular web framework
- Actix Web - Powerful, pragmatic, and extremely fast
- Rocket - Type-safe web framework with focus on usability
- Warp - Composable, fast web server framework
- And more...

### Database Support
- PostgreSQL with SQLx, Diesel, or SeaORM
- MySQL/MariaDB with full migration support
- SQLite for lightweight applications
- MongoDB for document storage
- Redis for caching
- ScyllaDB for high-performance NoSQL

### Advanced Features
- Docker multi-stage builds optimized for Rust
- GitHub Actions CI/CD workflows
- GraphQL with async-graphql or Juniper
- WebSocket support for real-time communication
- OpenTelemetry integration
- Frontend integration with HTMX, Leptos, or React
