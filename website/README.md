# Rust Blueprint Website

This is the official website for Rust Blueprint, built with Next.js and Panda CSS.

## 🚀 Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The site will be available at http://localhost:3000

### Building for Production

```bash
npm run build
npm run start
```

## 🎨 Tech Stack

- **Framework**: Next.js 14
- **Styling**: Panda CSS
- **UI Components**: Ark UI
- **Icons**: React Icons
- **Documentation**: Nextra

## 📁 Project Structure

```
website/
├── app/              # Next.js app directory
├── pages/            # Documentation pages
├── src/              # Source code
│   ├── components/   # React components
│   └── icons/        # Icon components
├── theme/            # Panda CSS theme configuration
├── public/           # Static assets
└── styled-system/    # Generated Panda CSS files
```

## 🎨 Theming

The website uses Rust's official colors:
- Primary: `#CE422B` (Rust red)
- Accent: `#F74C00` (Rust orange)
